# Grading Rubric — SE01WebEngineering

## Tools/AI setup
- 0 points: None
- 1 point: Basic (some config present)
- 3 points: Solid (gitignore, AGENTS.md, pylintrc, skills)

## Database schema
- 0 points: No database
- 1 point: Very basic DB (1-2 tables)
- 3 points: Solid (2+ tables, foreign keys)

## Business logic/views
- 0 points: No business logic
- 2 points: Basic scripts
- 3 points: Solid functionality

## Use of templates
- 0 points: None
- 1 point: Very basic (some templates exist)
- 3 points: Solid (complete separation of concerns: no webpage design in Python)

## User input
- 0 points: None
- 1 point: Basic (login/auth)
- 3 points: Solid (extensive capabilities)

## Rich interface/HTMX
- 0 points: None
- 1 point: Basic (responsive UI template used)
- 3 points: Solid (async interface, responsive and accessible UI)

## Tests, specs, documentation
- 0 points: None
- 1 point: Basic (readme, project info)
- 4 points: Extensive (tests, specs, documentation)

## Project deployment
- 0 points: No capability
- 1 point: Basic (can be deployed but no instructions given)
- 3 points: Solid (clear instructions to follow)
